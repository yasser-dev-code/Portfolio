Building a Responsive Portfolio with CSS Grid and Flexbox

October 15, 2023 5 min read Web Development CSS, Responsive Design, Layout

Creating a responsive portfolio website is essential for showcasing your work as a developer. With CSS Grid and Flexbox, you can build layouts that adapt beautifully to all screen sizes. In this tutorial, I'll walk you through the process of building a modern portfolio using these powerful CSS tools.

Why CSS Grid and Flexbox?

Before we dive into code, let's understand why these tools are so powerful:

  • CSS Grid is perfect for two-dimensional layouts (rows and columns)
  • Flexbox excels at one-dimensional layouts (either rows OR columns)
  • Both are fully responsive and work well together
  • They reduce the need for complex float-based layouts

Setting Up the Basic Structure

Let's start with the HTML structure for our portfolio:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Portfolio</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header class="header">
        <nav class="nav">
            <!-- Navigation content -->
        </nav>
    </header>
    
    <main class="main-content">
        <section class="hero">
            <!-- Hero section -->
        </section>
        
        <section class="projects">
            <!-- Projects grid -->
        </section>
        
        <section class="skills">
            <!-- Skills section -->
        </section>
    </main>
    
    <footer class="footer">
        <!-- Footer content -->
    </footer>
</body>
</html>

Creating the Hero Section with Flexbox

The hero section typically contains your introduction and a call-to-action. Here's how to create it with Flexbox:

.hero {
    display: flex;
    flex-direction: column;
    align-items: center;
    justify-content: center;
    min-height: 80vh;
    padding: 2rem;
    text-align: center;
}

@media (min-width: 768px) {
    .hero {
        flex-direction: row;
        text-align: left;
        justify-content: space-between;
    }
    
    .hero-content {
        flex: 1;
        max-width: 50%;
    }
    
    .hero-image {
        flex: 1;
        max-width: 50%;
    }
}

Building the Projects Grid with CSS Grid

For the projects section, CSS Grid is the perfect choice:

.projects-grid {
    display: grid;
    grid-template-columns: 1fr;
    gap: 2rem;
    padding: 2rem 0;
}

@media (min-width: 640px) {
    .projects-grid {
        grid-template-columns: repeat(2, 1fr);
    }
}

@media (min-width: 1024px) {
    .projects-grid {
        grid-template-columns: repeat(3, 1fr);
    }
}

.project-card {
    background: var(--card-bg);
    border-radius: 8px;
    overflow: hidden;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
    transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.project-card:hover {
    transform: translateY(-5px);
    box-shadow: 0 10px 20px rgba(0, 0, 0, 0.15);
}

Combining Grid and Flexbox for Complex Layouts

Sometimes you need both Grid and Flexbox. Here's an example of a skills section that uses both:

.skills-section {
    display: grid;
    grid-template-columns: 1fr;
    gap: 2rem;
}

@media (min-width: 768px) {
    .skills-section {
        grid-template-columns: repeat(3, 1fr);
    }
}

.skill-category {
    display: flex;
    flex-direction: column;
    padding: 1.5rem;
    background: var(--card-bg);
    border-radius: 8px;
}

.skill-list {
    display: flex;
    flex-wrap: wrap;
    gap: 0.5rem;
    margin-top: 1rem;
}

.skill-tag {
    padding: 0.25rem 0.75rem;
    background: var(--accent-color);
    color: white;
    border-radius: 20px;
    font-size: 0.875rem;
}

Responsive Navigation with Flexbox

A responsive navigation menu is crucial. Here's how to create one with Flexbox:

.nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem 2rem;
    background: var(--nav-bg);
}

.nav-menu {
    display: flex;
    list-style: none;
    gap: 2rem;
}

.nav-toggle {
    display: none;
    background: none;
    border: none;
    font-size: 1.5rem;
    cursor: pointer;
}

@media (max-width: 768px) {
    .nav-toggle {
        display: block;
    }
    
    .nav-menu {
        position: fixed;
        top: 70px;
        left: 0;
        width: 100%;
        flex-direction: column;
        background: var(--nav-bg);
        padding: 1rem;
        transform: translateY(-100%);
        opacity: 0;
        transition: all 0.3s ease;
    }
    
    .nav-menu.active {
        transform: translateY(0);
        opacity: 1;
    }
}

Conclusion

CSS Grid and Flexbox are powerful tools that make responsive design much easier. By combining them effectively, you can create complex layouts that work perfectly on all devices. Remember to:

  1. Use Flexbox for one-dimensional layouts (navigation, hero sections)
  2. Use CSS Grid for two-dimensional layouts (project grids, card layouts)
  3. Combine both for complex components
  4. Always test your layouts on multiple screen sizes

With practice, you'll be able to create beautiful, responsive portfolios that showcase your work effectively. Happy coding!
